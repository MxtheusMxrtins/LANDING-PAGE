Inicialmente comecei pelo HTML. HTML é uma abreviação de Hypertext Markup Language, ou seja, Linguagem de Marcação de Hipertexto. Resumindo, o HTML é uma linguagem usada para a publicação de conteúdo (texto, imagens, vídeos, áudio etc.) na web.

Após conhecer HTML ingressei em CSS. CCS define formatos e padrões estéticos dentro de uma página da web em um arquivo de códigos separado do HTML e/ou do JavaScript, chamados folha de estilo, facilitando a implementação destas modificações e trazendo mais facilidade ao trabalho do profissional desenvolvedor.

E atualmente conciliei os estudos com JavaScript. JavaScript é uma linguagem de programação que permite criar conteúdo dinâmico e interativo em páginas web. Com JavaScript, você pode manipular elementos do HTML, CSS, multimídia, gráficos, animações, e muito mais.
